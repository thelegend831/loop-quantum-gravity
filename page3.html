ADM metric:
$||g_{\mu\nu}|| = \overset{\mu(a)\downarrow\nu(b)\rightarrow}{\left[ \matrix{-\alpha^2 + \beta_c \beta^c & \beta_b \\ \beta_a & \gamma_{ab}} \right]}$
$||g^{\mu\nu}|| = \overset{\mu(a)\downarrow\nu(b)\rightarrow}{\left[ \matrix{-1/\alpha^2 & \beta^b / \alpha^2 \\ \beta^a / \alpha^2 & \gamma^{ab} - \beta^a \beta^b / \alpha^2} \right]}$
where $ \beta_b = \gamma_{ab} \beta^a = \gamma_{\mu b} \beta^\mu = g_{\mu b} \beta^b $ courtesy of the fact that $ g_{ab} = \gamma_{ab} $ and $ \beta^t = 0 $.

ADM metric determinant: $ g = \alpha \sqrt{\gamma} $
ADM spatial metric determinant: $ \gamma = det||\gamma_{ab}|| $

$n^\mu = \left[ \matrix{ 1/\alpha \\ -\beta^a/\alpha } \right] $

$n_\mu n^\mu = \left[ \matrix{-\alpha & 0} \right] \left[ \matrix{1/\alpha \\ -\beta^a/\alpha} \right] = -1$

$ \mathcal{L}_{\vec{n}} \gamma_{ab} = n^c \nabla_c \gamma_{ab} + \gamma_{cb} \nabla_a n^c + \gamma_{ac} \nabla_b n^c $
$ = n^c \nabla_c \gamma_{ab} + \gamma_{cb} \nabla_a n^c + \gamma_{ac} \nabla_b n^c $
$ = n^c \nabla_c (g_{ab} + n_a n_b) + (g_{cb} + n_c n_b) \nabla_a n^c + (g_{ac} + n_a n_c) \nabla_b n^c $
$ = n^c \nabla_c (n_a n_b) + \nabla_a n_b + n_c n_b \nabla_a n^c + \nabla_b n_a + n_a n_c \nabla_b n^c $
$ = n_a n^c \nabla_c n_b + n_b n^c \nabla_c n_a + \nabla_a n_b + \nabla_b n_a $
$ = ({\gamma^c}_a - \delta^c_a) \nabla_c n_b + ({\gamma^c}_b - \delta^c_b) \nabla_c n_a + \nabla_a n_b + \nabla_b n_a $
$ = {\gamma^c}_a \nabla_c n_b - \nabla_a n_b + {\gamma^c}_b \nabla_c n_a - \nabla_b n_a + \nabla_a n_b + \nabla_b n_a $
$ = {\gamma^c}_a \nabla_c n_b + {\gamma^c}_b \nabla_c n_a $
$ = {\gamma^c}_a {\gamma^d}_b \nabla_c n_d + {\gamma^c}_a {\gamma^d}_b \nabla_d n_c $
$ = {\gamma^c}_a {\gamma^d}_b (\nabla_c n_d + \nabla_d n_c) $
$ = 2 {\gamma^c}_a {\gamma^d}_b \nabla_{(c} n_{d)} $
$ \mathcal{L}_{\vec{n}} \gamma_{ab} = -2 K_{ab} $

Because $ \gamma_{ab} $ is symmetric, so is $ K_{ab} $

Lie derivative along shift vector of spatial metric:
$ \mathcal{L}_\vec\beta g_{ab} = \partial_c g_{ab} \beta^c + g_{cb} \partial_a \beta^c + g_{ac} \partial_b \beta^c $
using $ \partial_a \beta_b = \partial_a (\beta^c g_{cb}) = \partial_a \beta^c g_{cb} + \beta^c \partial_a g_{cb} $
so $ g_{cb} \partial_a \beta^c = \partial_a \beta_b - \beta^c \partial_a g_{cb} $
$ \mathcal{L}_\vec\beta g_{ab} = \partial_c g_{ab} \beta^c + \partial_a \beta_b - \beta^c \partial_a g_{cb} + \partial_b \beta_a - \beta^c \partial_b g_{ca} $
$ = \partial_a \beta_b + \partial_b \beta_a - \beta^c (\partial_a g_{cb} + \partial_b g_{ca} - \partial_c g_{ab} ) $
$ = \partial_a \beta_b + \partial_b \beta_a - 2 \beta^c \Gamma_{cba} $
$ = \partial_a \beta_b + \partial_b \beta_a - 2 \beta_c {\Gamma^c}_{ba} $
$ = \partial_a \beta_b - \beta_c {\Gamma^c}_{ba} + \partial_b \beta_a - \beta_c {\Gamma^c}_{ab} $
$ = \nabla_a \beta_b + \nabla_b \beta_a $
using $ \nabla^\perp_a v^b = {\gamma_a}^c \nabla_c v^b + n^b v^c K_{ac} $ for spatial $ v^b $
i.e. $ \nabla^\perp_a v^b = \nabla_a v^b + n_a n^c \nabla_c v^b + n^b v^c K_{ac} $
so $ \nabla_a v^b = \nabla^\perp_a v^b - n_a n^c \nabla_c v^b - n^b v^c K_{ac} $
so $ \nabla_a v_b = \nabla^\perp_a v_b - n_a n^c \nabla_c v_b - n_b v^c K_{ac} $
and $ \mathcal{L}_\vec\beta g_{ab} = \nabla^\perp_a v_b + \nabla^\perp_b v_a - n_a n^c \nabla_c v_b - n_b n^c \nabla_c v_a - n_b v^c K_{ac} - n_a v^c K_{bc} $

equivalence of Lie derivative along time vector and time partial derivative of spatial metric:
$ \mathcal{L}_\vec{t} \gamma_{ab} $
$ = \partial_a \gamma_{ab} t^a + \gamma_{cb} \partial_a t^c + \gamma_{ac} \partial_b t^c $

time derivative of spatial metric:
$ \dot{\gamma}_{ij} = \mathcal{L}_{\vec{t}}\gamma_{ij} $
$ = \mathcal{L}_{\alpha \vec{n} + \vec{\beta}} \gamma_{ij} $
$ = \alpha \mathcal{L}_{\vec{n}} \gamma_{ij} + \mathcal{L}_{\vec{\beta}} \gamma_{ij} $
$ = -2\alpha K_{ij} + \mathcal{L}_{\vec{\beta}} \gamma_{ij} $
$ \dot{\gamma}_{ij} = -2\alpha K_{ij} + 2 \nabla^\perp_{(i} \beta_{j)} $
